# 位域点击功能使用说明

## 功能概述

在SVD GUI Viewer中，现在可以直接点击寄存器位域图中的任意位域（field），在详细信息面板查看该位域的完整描述信息。

## 使用方法

### 1. 查看寄存器位域图

1. 在左侧树形结构中选择一个寄存器
2. 如果该寄存器包含位域信息，底部会自动显示位域图

### 2. 点击位域查看详情

![位域点击示例](C:/Users/Administrator/.gemini/antigravity/brain/0c1746f1-e86c-4090-9389-7b35bdf4315a/uploaded_image_1765261757441.png)

**操作步骤：**
- 鼠标移动到任意位域上，光标会变为手型 🖱️
- 位域边框会加粗显示（悬停效果）
- 点击位域，右侧详细信息面板会显示：
  - 位域名称
  - 位范围（如 Bits 15:8）
  - 位宽（占用多少位）
  - 访问权限（读写/只读/只写）
  - 位域描述信息
  - 所属寄存器及地址

### 3. 视觉反馈

**鼠标悬停时：**
- 光标变为手型（表示可点击）
- 位域边框加粗并变黑

**点击后：**
- 右侧详细信息面板更新
- 底部状态栏显示当前选中的位域

## 功能特点

✅ **交互式位域浏览** - 直接点击图形查看详情
✅ **即时反馈** - 鼠标悬停即显示可点击状态
✅ **详细信息** - 完整显示位域的所有属性
✅ **中英文对照** - 访问权限等信息提供中英文说明

## 详细信息面板内容

当点击一个位域时，详细信息面板会显示：

```
========================================
位域: AMCLK2_FREQ
========================================

位范围: Bits 15:8
位宽: 8 bit(s)
访问权限: 读写 (read-write)

描述:
Analog Module Clock Frequency Status Register
...（完整描述信息）...

所属寄存器: AMCLK_FREQ_STS
寄存器地址: 0x50006000

========================================
提示: 点击位域可查看详细信息
```

## 技术实现

- **点击事件绑定** - 每个位域矩形都绑定了点击事件
- **悬停效果** - 使用Canvas标签系统实现高亮效果
- **数据关联** - 位域数据与寄存器数据完整关联

## 适用场景

- 📖 学习和理解寄存器结构
- 🔍 快速查找特定位域的功能
- 📝 编写寄存器配置代码时参考
- 🐛 调试时查看位域定义
